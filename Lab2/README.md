# Лабораторная работа 2 — Алгоритм k-средних

Кластеризация точек на плоскости методом k-средних с оценкой качества классификации.

Подробное описание задания — [task.md](task.md).

## Структура проекта

```
Lab2/
├── task.md              # Описание задания
├── Makefile             # Сборка, запуск, тесты
├── cpp/
│   ├── kmeans.h         # Основная логика
│   ├── solution.cpp     # Точка входа
│   └── tests.cpp        # Тесты
├── python/
```

## Требования

- **C++**: `g++` с поддержкой C++11 или выше
- **Python**: `python3` (3.8+)
- **R**: `Rscript` (4.0+)

## Быстрый старт

```bash
# Собрать всё
make

# Запустить решение
make run-cpp
make run-python
make run-r

# Запустить тесты
make test          # C++ + Python + R
make test-cpp      # только C++
make test-python   # только Python
```

## Параметры запуска

| Параметр | Описание |
|----------|----------|
| n | Число точек |
| k | Число классов |
| m | Число кластеров |
| Режим | `gen` — генерация, `manual` — ручной ввод троек `(x y class)` |

## Пример

```
Число точек (n): 300
Число классов (k): 3
Число кластеров (m): 3
Ввод данных (gen/manual): gen

Генерация 300 точек...
Кластеризация...

=== Результаты кластеризации ===
Итераций:            8
Суммарное расстояние: 1823.45
Доля ошибок:         4.3333%
```

## Алгоритм

1. Инициализация `m` центроидов случайным выбором из входных точек
2. Назначение каждой точки ближайшему центроиду (евклидово расстояние)
3. Пересчёт центроидов как среднее координат точек кластера
4. Повтор шагов 2–3 до сходимости (сдвиг центроидов < ε)
5. Класс кластера = мажоритарный класс его точек
6. Доля ошибок = точки, чей класс ≠ классу кластера
