# Лабораторная работа 2 — Алгоритм k-средних

## Условие

Пусть задано **n** точек на плоскости, принадлежащих **k** различным классам.

Необходимо:
1. Разделить точки на **m** кластеров, минимизировав суммарное расстояние до центров.
2. Для каждого кластера определить, какому классу он принадлежит (по большинству).
3. Посчитать долю точек в кластерах, класс которых не совпадает с классом кластера.

## Входные данные

| Параметр | Описание |
|----------|----------|
| `n` | Число точек |
| `k` | Число классов |
| `m` | Число кластеров |
| Массив | `n` троек вида `(x, y, k)` — координаты и класс точки |

## Алгоритм

1. Случайно выбрать `m` начальных центроидов.
2. Назначить каждую точку ближайшему центроиду.
3. Пересчитать центроиды как среднее точек кластера.
4. Повторять шаги 2–3 до сходимости.
5. Определить класс каждого кластера по мажоритарному голосованию.
6. Вычислить долю ошибочно классифицированных точек.
