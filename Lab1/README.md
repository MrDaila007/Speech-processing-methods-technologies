# Лабораторная работа 1 — Дискретизация

Преобразование аудиосигнала под заданную частоту записи с квантованием по уровням.

Подробное описание задания — [task.md](task.md).

## Структура проекта

```
Lab1/
├── task.md              # Описание задания
├── Makefile             # Сборка, запуск, тесты
├── cpp/
│   ├── discretization.h # Основная логика
│   ├── solution.cpp     # Точка входа
│   └── tests.cpp        # Тесты
├── python/
│   ├── solution.py      # Решение
│   └── test_solution.py # Тесты
└── r/
    ├── solution.R       # Решение
    └── tests.R          # Тесты
```

## Требования

- **C++**: `g++` с поддержкой C++11 или выше
- **Python**: `python3` (3.8+)
- **R**: `Rscript` (4.0+)

## Быстрый старт

```bash
# Собрать всё
make

# Запустить решение
make run-cpp
make run-python
make run-r

# Запустить тесты
make test          # C++ + Python + R
make test-cpp      # только C++
make test-python   # только Python
make test-r        # только R

# Очистить артефакты
make clean
```

## Параметры запуска

| Параметр | Описание | Допустимые значения |
|----------|----------|---------------------|
| n | Секунды аудио | > 0 |
| Битрейт | Глубина квантования (бит) | 1–24 |
| Частота записи | Частота дискретизации (KHz) | 48, 96, 128, 176, 256 |
