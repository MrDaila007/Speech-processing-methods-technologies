# Лабораторная работа 1 — Дискретизация

## Условие

Пусть записано **n** секунд аудио.

### Заданные параметры

| # | Параметр | Описание |
|---|----------|----------|
| 0 | `n` | Количество секунд аудио |
| 1 | Количество значений | `10^6 * n` записанных значений частоты сигнала |
| 2 | Битрейт | Определяет количество ступенек дискретизации |
| 3 | Частота записи | 48 KHz, 96 KHz, 128 KHz, 176 KHz или 256 KHz |

### Задача

На основе полученных данных преобразовать исходные значения сигнала под указанную частоту записи звука:

1. Несколько значений попадают в один временной интервал — вычислить **среднее арифметическое**.
2. Округлить результат до **ближайшей ступеньки** дискретизации частоты.

### Дискретизация частоты

- Диапазон: **20 Hz — 20 000 Hz**
- Шаг равномерный, определяется битрейтом: `step = (20000 - 20) / (2^bitrate - 1)`

### Входные данные

Генерируются случайным образом в диапазоне **20 Hz — 20 000 Hz**.
